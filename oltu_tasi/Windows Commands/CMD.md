### Change Directory:
###### Örnek 1:  
```cmd
C:\Users\tanju> cd
```
> **Explanation:**
> Sadece `cd` yazarsak bulunduğunuz dizin yolunu verecektir. linux dağıtımlarında buna eş değer `pwd` komutudur.

### Kopyalama:
###### Söz dizimi(Syntax):
```cmd
C:\Users\tanju> copy [kaynak_dosya] [hedef_dizin]
```
